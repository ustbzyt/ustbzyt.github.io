Automated Sequence Generation and Data Prep Toolkit for LC-MS Intact Mass Analysis of Antibody Biologics

Organization: Bristol Myers Squibb

Date: May 2025

The Challenge:

High-throughput LC-MS intact mass analysis is essential for biologics screening, but its efficiency is hindered by tedious manual sequence setup and data prep steps, which slow workflows, increase errors, and limit throughput.

My Solution:

I developed a Python-based toolkit to significantly streamline this process, initially targeting our group's specific LC-MS workflow. The toolkit is also designed to be adaptable for other similar assays with some modifications:

  • Automated Sequence Generation:
    • The script utilizes plate map information provided by users (colleagues submitting samples for LCMS analysis).
    • It automatically generates a CSV file that can be directly imported into ThermoFisher Xcalibur software as an instrument run sequence.
    • Concurrently, it generates a second CSV file for intact mass data analysis using Byos from Protein Metrics.
  • User-Friendly Design:
    • Equipped with a GUI for ease of use by non-programmers.
    • Automatically installs required Python packages.
    • Offers customizable settings for naming conventions and data storage locations.

Impact:

  • Reduced a manual task from ~15 min to ~15 sec for a single sequence.
  • Minimized human errors.
  • Improved user experience.

Technologies Used: Python, GUI Development (Tkinter), CSV Processing

AI-Enhanced Intact Mass Data Communication: Automated Visualization and Expert-Level Comment Generation

Organization: Bristol Myers Squibb

Date: May 2025

The Challenge:

Following data processing with Protein Metrics software, a critical bottleneck is the manual interpretation and commenting on experimental results for each sample. This task, traditionally performed by specialized mass spectrometry professionals to ensure clarity for non-specialists, is highly time-consuming, especially in high-throughput environments. Additionally, effectively visualizing 96-well plate results for trend analysis remains a key challenge.

My Solution:

I developed a powerful program with two core functionalities:

  • AI-Driven Automated Comment Generation:
    • The program integrates report data exported from Protein Metrics with metadata from the plate map.
    • This consolidated information is sent to a cloud-based Large Language Model (LLM), currently OpenAI's GPT-4.1.
    • Through meticulous prompt engineering, the system leverages the LLM to automatically generate comments for each sample that emulate the insight of a biologics characterization expert. These comments are not only technically sound but also concise and easily understandable, greatly facilitating communication with non-specialist team members.
  • Interactive 96-Well Plate Heatmap Visualization:
    • The program processes report data exported from Protein Metrics software.
    • Based on this data, it automatically generates an interactive HTML-formatted 96-well plate heatmap.
    • This visualization clearly displays key metrics for each well, such as data quality and impurity levels, helping users quickly identify trends, assess sample quality, and make informed decisions.

Impact:

  • Automated a traditionally expert-driven task, significantly reducing the time and manual effort required by mass spectrometry professionals for writing individual sample comments.
  • Enhanced report clarity and consistency by providing high-quality, AI-generated interpretations accessible to a broader audience.
  • Enabled rapid identification of data trends and outliers through intuitive visualization.
  • Accelerated data-driven decision-making by making expert-level insights more readily available to project teams.

Technologies Used: Python, OpenAI API (GPT-4.1), Prompt Engineering, HTML, CSS, JavaScript (potentially for interactive visualization), Data Parsing.
