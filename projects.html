LC-MS抗体生物药高通量分析流程自动化

工作单位: Bristol Myers Squibb (根据您的Profile)

时间: 近期 (例如: 2023 - 2024年，请您填写具体时间)

面临的挑战:

抗体生物药（包括双特异性抗体、融合蛋白等）的高通量LC-MS完整分子量分析会产生大量数据。无论是仪器运行的样品序列设置，还是后续的数据分析准备，手动操作都非常耗时且容易出错。

我的解决方案:

我开发了一套基于Python的工具，以显著简化此流程：

  • 自动化序列生成:
    • 脚本根据用户（提交LCMS分析的同事）提供的孔板图（plate map）信息。
    • 自动生成一个CSV文件，该文件可直接导入ThermoFisher Orbitrap质谱仪的Xcalibur软件中作为仪器运行序列。这省去了手动输入样品名称、位置、仪器方法、文件保存路径和进样量等繁琐步骤。
    • 同时，自动生成第二个CSV文件，专门用于完整分子量质谱数据分析软件Protein Metrics (Byos)。样品在质谱仪上分析完毕后，只需将原始数据放入指定文件夹，即可在Byos中导入此CSV文件，几乎无需额外操作即可开始批量数据分析。
  • 用户友好的设计:
    • 为脚本配备了图形用户界面（GUI），方便没有编程背景的同事使用。
    • 集成了自动化功能，例如，如果同事的电脑上缺少必要的Python包，脚本会自动进行安装。
    • 提供了多种个性化设置选项，如质谱数据的命名格式、预设的质谱数据存放位置等，以适应不同用户和项目的需求。

项目影响:

  • 显著减少了LC-MS运行设置和数据分析初始化所需的人工操作和时间。
  • 最大限度地减少了样品信息录入过程中可能出现的人为错误。
  • 提高了完整分子量分析工作流程的整体效率和通量。
  • 改善了仪器操作员和数据分析师的用户体验。

所用技术: Python, GUI开发 (例如 Tkinter, PyQt), CSV文件处理。

Protein Metrics软件分析报告的深度自动化与可视化

工作单位: Bristol Myers Squibb (根据您的Profile)

时间: 近期 (例如: 2023 - 2024年，请您填写具体时间)

面临的挑战:

在使用Protein Metrics软件处理完数据后，为非质谱专业人士撰写全面且易于理解的实验结果注释，以及将96孔板的结果进行可视化以便于趋势分析，这些都是耗时且重要的环节。

我的解决方案:

我开发了一个功能强大的程序，包含两个核心功能：

  • AI驱动的自动化注释生成:
    • 程序能够整合从Protein Metrics导出的报告数据以及孔板图中的元数据（metadata）。
    • 将这些整合后的信息发送给云端的大型语言模型（LLM），目前使用的是OpenAI的GPT-4.1。
    • 通过精心的提示工程（prompt engineering）约束模型的输出，系统能自动生成专家级别、同时又简洁易懂的实验结果注释，极大地便利了与非专业团队成员的沟通。
  • 交互式96孔板热图可视化:
    • 程序能够处理从Protein Metrics软件导出的报告数据。
    • 基于这些数据，自动生成一个交互式的HTML格式的96孔板热图（heatmap）。
    • 这种可视化方式清晰地展示了每个孔的数据质量、杂质水平等关键指标，帮助用户快速发现趋势、评估样品质量，并做出明智决策。

项目影响:

  • 大幅减少了手动撰写报告注释的时间和精力。
  • 提高了报告对更广泛受众的清晰度和一致性。
  • 通过直观的可视化，实现了数据趋势和异常值的快速识别。
  • 加速了项目团队基于数据的决策过程。

所用技术: Python, OpenAI API (GPT-4.1), 提示工程 (Prompt Engineering), HTML, CSS, JavaScript (可能用于交互式可视化), 数据解析。
